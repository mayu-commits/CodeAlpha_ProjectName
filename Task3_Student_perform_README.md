 Dataset

The dataset used in this project:

File: ResearchInformation3.csv

It contains student-related information such as:

GPA
HSC results
SSC results
Attendance
Preparation time
Department
Gender
Hometown
Family income

This dataset was used to analyze academic performance trends and resource factors affecting students.

 Dashboard Features

The dashboard provides several key insights:

 Key Performance Indicators (KPIs)
Average GPA: 3.18
Average HSC: 4.10
Average SSC: 4.80
Average Attendance: 71.09%
Total Students: 129
 Visualizations Included

 Average GPA & HSC by Computer Usage

Shows how computer usage relates to academic results.

 Attendance vs GPA Analysis

Scatter plot showing relationship between student attendance and GPA.

 Average GPA & HSC by Gender

Comparison of academic performance between male and female students.

 Student Details Table

Displays preparation time, department, attendance category, and GPA.
 Interactive Filters (Slicers)

Users can explore the data using filters such as:

Department
Gender
Hometown (City / Village)
Income Level

These slicers allow dynamic exploration of student performance across different groups.
